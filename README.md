# Verdi Desouk POS (Flask + SQLite)

نظام كاشير عربي كامل مع واجهة RTL، باركود، فواتير حرارية، وإدارة شيفتات.

## التشغيل السريع (Windows PowerShell)
```
cd C:\Users\sayed-elhaddad\Desktop\v-system
python -m venv .venv
.\.venv\Scripts\activate
pip install -r requirements.txt
python app.py
```
الخدمة تعمل على http://localhost:5000

حساب افتراضي: اسم المستخدم `admin` وكلمة المرور `admin123` (يُنصح بتغييرها فورًا).

## المجلدات
- `app/` الكود الأساسي (Flask، النماذج، المسارات، القوالب)
- `app/templates/` قوالب Jinja2 عربية RTL
- `app/static/` ملفات CSS/JS والباركود المولّد
- `instance/verdi.db` قاعدة بيانات SQLite (تُنشأ تلقائيًا)

## المزايا
- واجهة عربية RTL نظيفة بBootstrap RTL + jQuery
- مصادقة جلسات وأدوار (مشرف/كاشير)
- إدارة منتجات مع باركود فريد وتوليد صورة باركود
- شاشة كاشير سريعة تدعم إدخال الباركود من لوحة المفاتيح
- فواتير حرارية جاهزة للطباعة تشمل الشعار
- تنبيهات مخزون منخفض/منتهي
- عملاء بسيطون مع إجمالي مشتريات
- إدارة شيفتات (فتح/إغلاق/فرق نقدية)
- تقارير يومية/أسبوعية/شهرية + الأكثر مبيعًا + ملخص شيفتات

## مسارات رئيسية
- تسجيل الدخول: `/login`
- لوحة التحكم: `/`
- المنتجات: `/products`
- المخزون والتنبيهات: `/inventory`
- نقطة البيع: `/pos`
- العملاء: `/customers`
- التقارير: `/reports`
- الشيفتات: `/shifts`
- الإعدادات (مسار اللوجو): `/settings`
- API البحث عن منتج: `/api/products/search?q=`
- API إنشاء فاتورة: `POST /api/sale`
- عرض الفاتورة/الطباعة: `/invoice/<sale_id>`

## تخصيص اللوجو
- ضع المسار في صفحة الإعدادات (مثال: `static/img/logo.png`) ليظهر في تسجيل الدخول واللوحة والفاتورة.

## ملاحظات أمنية
- غيّر `SECRET_KEY` وبيانات المدير الافتراضي.
- لا تترك قاعدة البيانات مفتوحة للكتابة في بيئة مشتركة.
